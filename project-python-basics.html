Python Basics — fundamenten

Deze module legde de fundamenten voor alle verdere Python- en securityprojecten. Ik bouwde kleine consoleprogramma's om de logica van programmeren echt in de vingers te krijgen.

Wat kwam er in deze cursus aan bod?

  • Basisconcepten: variabelen, datatypes en operators.
  • Beslissingsstructuren met if/elif/else.
  • Lussen met for en while om taken te herhalen.
  • Functies schrijven om logica op te splitsen en herbruikbaar te maken.
  • Werken met lijsten, dictionaries en eenvoudige foutafhandeling.

Wat wordt er nu van mij verwacht?

  • Dat ik leesbare en gestructureerde basiscode in Python kan schrijven.
  • Dat ik problemen in kleine stappen kan opdelen en in code kan vertalen.
  • Dat ik code van anderen kan begrijpen en waar nodig aanpassen.
  • Dat ik comfortable ben met de command line en een IDE zoals VS Code.
  • Dat ik klaar ben om meer gevorderde Python-toepassingen te bouwen.
