Tools of the Trade — Linux & SQL

In dit project combineer ik de kracht van Linux en SQL om grote hoeveelheden data sneller te begrijpen. Denk aan logbestanden doorzoeken, filters bouwen en patronen vinden in miljoenen rijen.

Wat kwam er in deze cursus aan bod?

  • Navigeren en werken in de Linux-shell (bestanden, rechten, processen).
  • Gebruik van veelvoorkomende tools zoals grep, awk en sed.
  • SQL-basics: SELECT, WHERE, GROUP BY, ORDER BY, JOIN.
  • Data uit logbestanden omzetten naar een vorm die met SQL te analyseren is.
  • Combineren van command line en SQL in praktische workflows.

Wat wordt er nu van mij verwacht?

  • Dat ik vlot met de Linux-command line kan werken.
  • Dat ik SQL-queries kan schrijven om gerichte vragen op data te stellen.
  • Dat ik logbestanden kan filteren en samenvatten op basis van patronen.
  • Dat ik efficiënt kan schakelen tussen terminal en database-tools.
  • Dat ik begrijp welke tools ik wanneer moet inzetten.
